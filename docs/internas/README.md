# Documentacion Interna

Aca va toda la documentacion de la empresa relacionada con el proceso de modernizacion.

## Que poner aca
- PDFs de arquitectura, decisiones tecnicas, procesos.
- Exports de Word/Google Docs con procedimientos, runbooks, etc.
- Cualquier documento interno que sirva como fuente para la tesis.

## Organizacion sugerida
- Nombrar archivos de forma descriptiva: `arquitectura-openshift-v1.pdf`, `runbook-deploy-iis.docx`
- Si hay muchos, crear subcarpetas por tema: `arquitectura/`, `procesos/`, `decisiones/`

> IMPORTANTE: No subir documentos con informacion sensible (credenciales, IPs internas, datos de clientes). Sanitizar antes de commitear.
