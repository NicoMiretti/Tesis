# Diagramas

Diagramas de arquitectura, flujos, comparativas antes/despues.

## Que poner aca
- Screenshots de la infra actual y anterior.
- Diagramas de arquitectura (draw.io, Excalidraw, Lucidchart, etc.).
- Capturas de dashboards, pipelines, Argo CD, Developer Hub.
- Fotos de pizarras si las hay.

## Formato
- Guardar los editables (.drawio, .excalidraw) junto con los exports (.png, .svg).
- Nombrar descriptivamente: `arquitectura-antes.png`, `flujo-gitops-argocd.drawio`
