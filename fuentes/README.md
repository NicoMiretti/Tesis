# Fuentes Bibliograficas

Material academico y tecnico para citar en la tesis.

## Estructura

- `libros/` - Libros tecnicos (Kubernetes in Action, GitOps and Kubernetes, etc.)
- `papers/` - Papers academicos y de investigacion.
- `articulos/` - Articulos de blog, conferencias, charlas relevantes.

## Formato sugerido
Crear un archivo `referencias.md` en esta carpeta con todas las citas en formato APA o el que pida la universidad.
